# Industrial-Anomaly-Detection-
Deep learning based industrial anomaly detection benchmark on MVTec AD using ResNet50_AE, PatchCore, DINOv2_KNN, Swin_PatchCore, and ViT. Includes robustness testing under real-world distortions, anomaly heatmaps, AUROC/AP evaluation, and comparative analysis using PyTorch on Kaggle GPU.
